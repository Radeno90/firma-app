# Firma – CEO Simulácia PWA

## Obsah
- `index.html` – hlavná aplikácia
- `manifest.json` – PWA manifest
- `sw.js` – service worker (offline podpora)
- `icons/` – ikony aplikácie (treba doplniť)

## Nasadenie na GitHub Pages (zadarmo)

1. Vytvor účet na github.com
2. Klikni "New repository" – pomenuj ho napr. `firma-app`
3. Nahraj všetky súbory (index.html, manifest.json, sw.js)
4. Choď do Settings → Pages → Source: "Deploy from branch: main"
5. Appka bude dostupná na: `https://TVOJE-MENO.github.io/firma-app`

## Inštalácia na telefón

### iPhone (Safari):
1. Otvor URL v Safari
2. Klepni na ikonu zdieľania (štvoreček so šípkou)
3. Vyber "Pridať na plochu"

### Android (Chrome):
1. Otvor URL v Chrome
2. Klepni na tri bodky (menu)
3. Vyber "Pridať na plochu" alebo "Nainštalovať aplikáciu"

## Funkcie
- Boardroom s 5 AI asistentmi
- Každý asistent má vlastnú osobnosť a odbornosť
- Chat s každým asistentom zvlášť
- API kľúč uložený len lokálne v zariadení
- Funguje offline (základná vrstva)

## Ikony
Pre správne zobrazenie ikony appky vytvor:
- `icons/icon-192.png` (192x192px)
- `icons/icon-512.png` (512x512px)

Môžeš použiť akýkoľvek online PNG generátor.
